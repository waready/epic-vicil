# Roadmap

## Fase 1: MVP Institucional

- Login API con Sanctum
- Roles y permisos
- Facultades y programas
- Ciclos de acreditacion
- Criterios, subcriterios y requerimientos
- Checklist de evidencias
- Carga de archivos
- Versiones
- Observacion, validacion y aprobacion
- Dashboard basico
- Exportacion ZIP sincrona

## Fase 2: Portafolio de Cursos

- Planes de estudio avanzados
- Ofertas por semestre
- Silabos, evaluaciones, rubricas y trabajos
- Alertas por curso incompleto
- Responsables por portafolio

## Fase 3: Calidad y Resultados

- Atributos del graduado
- Indicadores
- Rubricas
- Mediciones directas e indirectas
- Consolidado de resultados

## Fase 4: Mejora Continua

- Planes de mejora
- Acciones y responsables
- Seguimiento
- Evidencias de implementacion
- Evaluacion de impacto

## Fase 5: Infraestructura y Exportacion Avanzada

- Laboratorios
- Equipos
- Software
- Mantenimiento
- Jobs asincronos para ZIP grandes
- URLs firmadas de descarga
- Integracion con storage S3 compatible
