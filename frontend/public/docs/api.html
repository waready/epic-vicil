Documentacion API

Sistema de Gestion de Acreditacion y Evidencias Academicas

Frontend

https://acreditacion.waready.org.pe

Base API

https://waready.org.pe/api

Autenticacion

Bearer token con Laravel Sanctum.

Rutas Principales

Modulo Rutas
Auth POST /auth/login, GET /auth/me, POST /auth/logout, POST /auth/change-password
Perfil GET /profile, PUT /profile
Catalogos GET /faculties, /programs, /study-plans, /semesters, /courses, /teachers
Acreditacion GET /accreditation-models, /accreditation-cycles, /accreditation-criteria, /evidence-requirements, /evidence-tasks/catalog
Mis evidencias GET /my/evidence-tasks, POST /evidence-tasks/{id}/submissions
Evidencias GET /evidences, POST /evidences, GET /evidences/{id}, POST /evidences/{id}/versions
Revision POST /evidences/{id}/observe, /validate, /approve
Subida directa POST /uploads/direct/presign, POST /uploads/direct/complete
Dashboard GET /dashboard/summary, /progress-by-criterion, /progress-by-program, /pending-by-teacher
Exportacion POST /exports/evidences-zip

Pruebas Rapidas

Health

curl https://waready.org.pe/api/health

Login

curl -X POST https://waready.org.pe/api/auth/login \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  --data '{"email":"usuario@dominio.edu.pe","password":"TU_CONTRASENA"}'

Consultar usuario autenticado

curl https://waready.org.pe/api/auth/me \
  -H "Authorization: Bearer TOKEN" \
  -H "Accept: application/json"

Cambiar contrasena

curl -X POST https://waready.org.pe/api/auth/change-password \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  --data '{"current_password":"CONTRASENA_ACTUAL","password":"NuevaClave123","password_confirmation":"NuevaClave123"}'
